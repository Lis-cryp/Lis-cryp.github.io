How the service works

  1. Create an account and sign in to the creator workspace.
  2. Click Connect Pinterest and approve requested access.
  3. Return to the secure callback URL to finish the connection.
  4. Manage drafts and publish content only after explicit user action.

Business presentation

Dreamer Studio is presented as a real creator service, not a single-link landing page. The product includes a homepage, support information, account area, legal pages, and a documented Pinterest connection flow for review purposes.

Pinterest connection flow

The service uses a secure OAuth integration. Users start from the login page, review the Pinterest connect button, are redirected to Pinterest authorization, and then return to a dedicated callback URL.

Redirect URI example:
https://your-domain.com/auth/pinterest/callback

For local review this project also includes a static callback page at /auth/pinterest/callback/.
